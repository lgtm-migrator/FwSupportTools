FieldWorks data model

FieldWorks is a suite of integrated language research software applications. This help file contains the documentation for the FieldWorks data model. The data model is expressed in the Unified Modeling Language. For more information about UML refer to: http://www.uml.org/

To use this help file, click on the Diagrams link in the top left frame. Select a diagram from the bottom left frame. Start with the ***Legend diagram for some brief explanation.

Classes can be clicked on in class diagrams to get full documentation on the class.

For more information, contact Larry Hayashi at larry_hayashi@sil.org.

Diagrams and documentation produced using MagicDraw ( http://www.magicdraw.com ).
This model is used to generate a SQL database.
